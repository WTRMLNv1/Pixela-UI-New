# Pixela üñåÔ∏è

**Pixela** is a desktop application built with **Python** and **CustomTkinter** that allows users to **track habits, create graphs, and visualize their productivity** in a sleek, interactive interface. Perfect for anyone who loves data, visuals, or just wants to see their progress over time!  
This project is built on the [Pixela API](https://pixe.la), a habit-tracking API that allows users to manage graphs, add pixels, and visualise progress. Pixela provides a full GUI, allowing users to perform all of this without writing the code typically required by the API.

---

## Features ‚ú®

- **User Accounts:** Login/Sign up system with persistent credentials.  
- **Graph Creation:** Easily create custom graphs with units, types, and colours.  
- **Add Pixels:** Track daily activities or data points in your graphs.  
- **Dynamic Heatmap:** Visualise your data at a glance.  
- **Custom Accent Colours:** Personalise the app with your favourite colours using an interactive HSV colour picker.  
- **Settings Screen:** Set your preferred graph for display and manage accent colours.  
- **Responsive UI:** Clean, modern, and consistent design.  

---
## Who is this for?
Ideal for students, developers, or anyone who wants visual motivation for daily habits.

## Installation ‚ö°

1. Clone this repo:  
```bash
git clone https://github.com/yourusername/Pixela.git
cd Pixela
```
2. Install Dependencies
```bash
pip install -r requirements.txt
```
Or Download manually:
```bash
pip install customtkinter
pip install Pillow
pip install matplotlib
pip install numpy
pip install requests
```
3. Run the app
```bash
python UI.py
```
